This plugin adds a payment gateway for Paytrail.

== Description ==

EDD Paytrail Gateway is add-on plugin for [Easy Digital Downloads](http://wordpress.org/extend/plugins/easy-digital-downloads/) plugin. You need to install
that first. It enables Paytrail gateaway in your shop.

== Installation ==

1. Upload `edd-paytrail` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
